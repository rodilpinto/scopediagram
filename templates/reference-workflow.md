# Fluxo de Referência

Use o PowerPoint fornecido como fonte de verdade para os próximos ajustes visuais.

## Fluxo

1. Compare os slides de subprocesso gerados com o arquivo de referência.
2. Ajuste coordenadas, tamanhos de fonte e preenchimentos em `ppt.py`.
3. Mantenha o schema estável, salvo mudança real no significado de negócio.
4. Prefira alterar constantes de layout em vez de criar lógica ad hoc.
5. Não introduza dependência de SmartArt.

## Prioridades de ajuste

1. Largura das colunas e espaçamento vertical
2. Tamanho da fonte para listas longas de reguladores e recursos
3. Legibilidade das caixas de evento
4. Equilíbrio visual entre slides com pouco e muito conteúdo

## Quando adicionar um template `.pptx` real

Adicione um template somente se houver capa, rodapé, logomarca ou identidade visual fixa que precise ser preservada com exatidão. Para o corpo do diagrama, a geração por formas continua sendo a abordagem preferida.
